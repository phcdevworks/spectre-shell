## [0.0.1] - 2026-02-05

### Added

- Initial release: Thin, framework-agnostic app shell for Spectre platform.
- Bootstrap function for initializing apps with routing and styling.
- Integration with `@phcdevworks/spectre-shell-router` for client-side routing.
- Integration with `@phcdevworks/spectre-ui` for styling framework.
- Auto-import of Spectre design tokens and UI styles.
- TypeScript configuration with strict type checking.
- Package exports configuration for ES module compatibility.
- Repository boilerplate including MIT license, README, VS Code workspace, and project configuration files.
